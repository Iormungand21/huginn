# Automation Queue

Tracked assets for autonomous Claude worker runs in this repository.

- `backlog.tsv`: repo-local machine-readable task queue (for this repo only)
- `tasks/*.md`: detailed task briefs
- `runtime/`: logs/locks/prompts generated by runner (ignored by git)
- `templates/muninn-backlog.tsv`: seed template to copy into the `muninn` repo

Run one task:

```bash
scripts/claude-next-task.sh
```

Run multiple tasks in sequence:

```bash
scripts/claude-next-task.sh --loop --max-tasks 3
```
